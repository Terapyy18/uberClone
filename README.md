# uberclone
Ceci est un template Expo Supabase Redux et Auth 

créer un projet Expo - Supabase

Récuperer les clées nécéssaires dans le dashboard Supabase et les mettre dans le fichier .env

EXPO_PUBLIC_SUPABASE_URL=""
EXPO_PUBLIC_SUPABASE_KEY=""
+ lié le projet à votre compte supabase avec : eas init --id ID DU PROJET

installer les dépendances nécéssaires:

npm install
